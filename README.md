# skill-project-tracker
MVP project for practicum — Skill/Project Tracker with AI suggestions.

This is my Practicum 1 project MVP.

## Features (MVP)
- User login & logout
- Dashboard with simple analytics
- Skill logging
- Project logging
- AI-based project/skill suggestions

## Tech Stack
- Backend: Java (Spring optional later)
- Frontend: HTML/CSS/JS (React optional later)
- Database: SQL
